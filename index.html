What is an Array?

An array is a collection of variables of the same data type referred to by a common name. Elements are stored in contiguous memory locations, where the lowest address corresponds to the first element and the highest to the last.

Index always starts from 0 in an array. The last element is at index size − 1.
An array is also known as a subscripted variable.
Before using an array, its type and dimension must be declared.
A specific element is accessed by its index (subscript).

One-Dimensional Arrays

A one-dimensional array (1-D array) is a variable that represents a list of items using only one index (subscript). It is the simplest form of an array.

Declaration Syntax

C Syntax
type arr_name[size];

// Examples:
int marks[30]; // 30 integers
float press[10]; // 10 floats
int b[100], x[27]; // two arrays at once

type

Any valid C data type: int, float, char, etc. All elements must be the same type.

size

Specifies how many elements the array can hold. The brackets [ ] tell the compiler we are dealing with an array.

Initializing Arrays in C

C — Initialization

Accessing Array Elements

Elements are accessed using a subscript. The subscript specifies the element's position. All elements are numbered starting with 0, so arr[2] is the third element, not the second.

C — Accessing Elements
int arr[] = {1, 2, 3, 4, 5};

int val = arr[2]; // val = 3 (third element)

// Array elements behave like normal variables
c[0] = 3;
printf("%d", c[0]); // prints 3

// Operations in subscript (if x == 3)
c[5-2] == c[3] == c[x] // all refer to same element

1-D Array Input / Output

C — Array I/O
#include <stdio.h>
int main() {
int values[5];
printf("Enter 5 integers: ");

// Taking input and storing in the array
for(int i = 0; i < 5; ++i) {
scanf("%d", &values[i]);
}

// Printing elements of the array
for(int i = 0; i < 5; ++i) {
printf("%d\n", values[i]);
}
return 0;
}

Sum of Array Elements

C — Sum of Array
#include <stdio.h>
#define SIZE 12
int main(void) {
int a[12] = {1,3,5,4,7,2,99,16,45,67,89,45};
int i, total = 0;

for(i = 0; i < SIZE; i++) {
total += a[i]; // accumulate sum
}

printf("Total = %d\n", total);
// Output: Total of array element values is 383
return 0;
}

Find Maximum and Minimum Element

C — Max & Min
#include <stdio.h>
int main() {
int arr1[100], i, max, min, n;

printf("Number of elements: "); scanf("%d", &n);
for(i = 0; i < n; i++) scanf("%d", &arr1[i]);

max = min = arr1[0]; // initialise with first element

for(i = 1; i < n; i++) {
if(arr1[i] > max) max = arr1[i]; // update max
if(arr1[i] < min) min = arr1[i]; // update min
}
printf("Max = %d, Min = %d\n", max, min);
return 0;
}

Practice: Insert an Element in an Array

📊 Insert at position 5 — shift elements forward
C — Insert Element at Specific Position
#include <stdio.h>
int main() {
int arr[100] = {0};
int i, x, pos, n = 10;

for(i = 0; i < n; i++) scanf("%d", &arr[i]);

x = 50; // element to insert
pos = 5; // position (1-indexed)
n++; // increase size

// Shift elements forward from the end
for(i = n; i >= pos; i--)
arr[i] = arr[i - 1];

arr[pos - 1] = x; // insert x at pos

for(i = 0; i < n; i++) printf("%d ", arr[i]);
return 0;
}

Linear Search

Linear search checks each element of the array one by one from the beginning until the target element is found or the array ends. Also called sequential search.

Algorithm Steps

1
Read the search element from the user.
2
Compare the search element with the first element in the list.
3
If both are matched, display "Given element is found!!!" and terminate.
4
If not matched, compare search element with the next element in the list.
5
Repeat steps 3 and 4 until the search element is compared with the last element.
6
If the last element also doesn't match, display "Element is not found!!!" and terminate.

C — Linear Search Implementation
#include <stdio.h>
int main() {
int array[100], search, c, n;
printf("Enter number of elements\n"); scanf("%d", &n);
printf("Enter %d integers\n", n);
for(c = 0; c < n; c++) scanf("%d", &array[c]);

printf("Enter number to search\n"); scanf("%d", &search);

// Linear search loop
for(c = 0; c < n; c++) {
if(array[c] == search) {
printf("%d found at location %d\n", search, c+1);
break;
}
}
if(c == n)
printf("%d not in array\n", search);
return 0;
}

Binary Search

Binary search works on a sorted array by repeatedly dividing the search interval in half. It compares the target with the middle element and eliminates half the array each time, making it much faster than linear search.

⚠️ Pre-condition

The array must be sorted (ascending or descending) before binary search can be applied.

⚡ Efficiency

Binary search is O(log n) — for 1,000,000 elements, it needs at most ~20 comparisons!

C — Binary Search Implementation
#include <stdio.h>
void main() {
int n, i, arr[50], search, first, last, middle;
printf("Enter total number of elements: "); scanf("%d", &n);
printf("Enter %d numbers: ", n);
for(i = 0; i < n; i++) scanf("%d", &arr[i]);

printf("Enter a number to find: "); scanf("%d", &search);

first = 0;
last = n - 1;
middle = (first + last) / 2;

while(first <= last) {
if(arr[middle] < search) {
first = middle + 1; // search right half
} else if(arr[middle] == search) {
printf("%d found at location %d\n", search, middle+1);
break;
} else {
last = middle - 1; // search left half
}
middle = (first + last) / 2;
}
if(first > last)
printf("Not found! %d not in list.\n", search);
}

Linear vs Binary Search — Comparison

Feature Linear Search Binary Search
Array requirement Works on unsorted arrays Array must be sorted
Time complexity O(n) — worst case O(log n) — much faster
Best case O(1) — element at index 0 O(1) — element is the middle
Method Sequential, one by one Divide and conquer
Suitable for Small or unsorted data Large sorted datasets
Implementation Simple — one loop Slightly more complex
